# Laboratorio de Funciones Matemáticas Web

Aplicación web didáctica para graficar funciones algebraicas, exponenciales, logarítmicas y trigonométricas.

El proyecto toma apuntes de cálculo diferencial y los transforma en una herramienta interactiva para analizar dominio, rango, simetría, transformaciones y operaciones entre funciones.

---

## Objetivo del proyecto

Crear una herramienta visual para practicar familias de funciones y observar cómo cambian sus gráficas al modificar parámetros.

El sistema permite trabajar con:

- Rectas que pasan por el origen.
- Funciones constantes.
- Rectas verticales.
- Funciones algebraicas.
- Funciones raíz.
- Funciones valor absoluto.
- Funciones exponenciales.
- Funciones logarítmicas.
- Funciones trigonométricas.
- Operaciones entre funciones.
- Transformaciones y reflexiones.
- Exportación de gráficas como PNG.

## Enfoque académico

Este laboratorio convierte apuntes de funciones matemáticas en una web app interactiva.

El usuario puede seleccionar una familia de funciones, modificar parámetros, observar la gráfica, consultar dominio/rango y generar una tabla de valores.
